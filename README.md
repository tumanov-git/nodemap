# Ref1

MVP веб-приложения для визуальной карты понятий по теме онтологий в имитационном моделировании.

## Стек

- `React`
- `Vite`
- `@xyflow/react` для доски с узлами и связями

Такой набор уже выглядит современно, но остается простым для учебного проекта.

## Что уже есть

- визуальная доска с узлами и связями;
- пять типов понятий: объект, процесс, ресурс, событие, состояние;
- редактирование выбранного узла через инспектор;
- удаление узлов и связей;
- экспорт и импорт в `JSON`;
- сохранение состояния в `localStorage`;
- демо-сценарий на примере очереди в поликлинике.

## Запуск

```bash
cd "/Users/izotop/md thing/ref1"
npm install
npm run dev
```

По умолчанию Vite поднимет локальный адрес, обычно `http://localhost:5173`.

## Идея развития

- подписи на связях;
- готовые шаблоны предметных областей;
- экспорт доски в изображение;
- режим презентации для защиты.
